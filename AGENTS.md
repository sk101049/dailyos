# DailyOS AI Engineering Manual

## Project
DailyOS is an AI-powered daily workspace for insurance professionals.

## Roles
- CEO: Product direction
- GPT CTO: Planning, architecture, code review
- Codex: Implement one GitHub Issue at a time

## Development Rules
1. One Issue = One Goal
2. One Commit = One Issue
3. Keep main branch deployable
4. No features outside the current Issue
5. Read docs/PROJECT.md, ROADMAP.md and TASKS.md before coding.

## Coding Standards
- TypeScript
- Next.js App Router
- Tailwind CSS
- shadcn/ui
- Small reusable components
