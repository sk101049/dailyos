# DailyOS Architecture

## Principles
- Modular feature-based structure.
- UI first, backend second.
- AI integrations behind service interfaces.

## Layers
1. app/ - routing and layouts
2. features/ - business modules
3. components/ - reusable UI
4. lib/ - shared services
5. types/ - shared models

## Planned Modules
- Dashboard
- Content Studio
- CRM
- AI Assistant
- Automation

## Roadmap
Build UI -> Connect data -> Add AI -> Add automation.
