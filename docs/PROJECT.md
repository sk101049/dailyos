# Project

## Vision

DailyOS is a personal AI insurance workspace for organizing daily insurance work, client follow-up, content creation, and assistant-supported planning in one focused place.

The vision is to make each workday easier to start by giving the user one clear place to see priorities, plan action, and move client and content work forward.

## Target User

DailyOS is for an insurance professional who wants a simple daily operating system for deciding what to work on next, managing client relationships, and preparing content without switching between scattered tools.

## V1 Goal

The V1 goal is to help the user open one workspace each day and clearly see the next actions across client work, content work, and AI-assisted planning.

## Core Modules

- Dashboard
- Content Studio
- CRM
- AI Assistant
