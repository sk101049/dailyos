# Roadmap

## Sprint 0: Project Foundation

Create the initial documentation and planning structure for DailyOS.

## Sprint 1: Dashboard

Build the first daily workspace view so the user can see priorities and next actions.

Status: Complete in MVP.

## Sprint 2: CRM

Create the first client relationship workflow for tracking contacts, notes, and follow-ups.

Status: Complete in MVP with placeholder client workflow.

## Sprint 3: AI Content

Add the first AI-assisted content planning and drafting workflow.

Status: Complete in MVP with OpenAI script generation, GPT Mode, GPT output paste, editable preview, and Script Library.

## Sprint 4: Video Workflow

Plan and build a workflow for turning insurance content ideas into video-ready assets.

Status: Complete in MVP with Thumbnail Prompt Builder and Storyboard Builder.

## Sprint 5: Automation

Introduce automation for routine follow-ups, reminders, and daily planning support.

Status: Not started. Publishing Center was added first as a local-first beta workflow.

## Beta 1: MVP Stabilization

Validate the existing Dashboard, Content Studio, CRM, and Publishing Center workflows, then document the first usable beta.

Status: Complete for v0.1.0-beta.
