# DailyOS

DailyOS is a personal AI insurance workspace that helps the user open one daily workspace and know what to do next.

Current version: 0.1.0-beta

## Current MVP

- Dashboard for daily priorities and assistant suggestions.
- Content Studio with GPT Mode, GPT output paste, editable script preview, Script Library, Thumbnail Prompt Builder, and Storyboard Builder.
- CRM workspace for client follow-up planning.
- Publishing Center for local-first short-form video publishing status.
